21 /*
22 File-Format:
23 
24 Address
25 0..2 filemarking "ZFS"
26 3..4 version of fileformat (WORD i.e. 0201h = 2.1 )
27 5..6 address of start of data = headersize (WORD i.e. 4096d)
28 7 reserved (BYTE = 0)
29 8.. Start of items (see below)
30 
31 
32 Items (each one has at least 4DWORD's = 16Byte)
33 0..1 index (see below)
34 2 len of value (4=DWORD, 8=double, 9=two floats, or stringlength if type=1)
35 3 type (0=solid value, 1=string/array ) from fileversion 2.1
36 4..15 value if it is not a string/array
37 ... maybe more if string/array used. Setting ends always at a boundary of 16Bytes. Sinze fileversion 2.1
38 
39 last item has index 0xffff
40 
41 */
