Detailed Description

The 4x4 matrix that is used as a transformation matrix between 3d-coordinate systems.

Transformations Matrix

How the matrix translates into an 1D array m[16] Matrix Members: Array Indizes: a1 b1 c1 dTranslateX 0 1 2 3 a2 b2 c2 dTranslateY 4 5 6 7 a3 b3 c3 dTranslateZ 8 9 10 11 a3 b4 c4 scale 12 13 14 15

ZFTRMatrix::World(x) is like m*x
